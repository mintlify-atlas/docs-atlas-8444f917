Memory Pak launches directly into your collection — no sign-up, no configuration file, no internet connection required. This guide walks you through the core workflow: finding a console, marking it as owned, locating a game, and exporting a backup of your collection state.

## App layout

When Memory Pak opens you'll see three areas:

- **Sidebar** (left) — four tabs: Consoles, Games, LEGO Dimensions, Skylanders. On screens narrower than 700 px, tap ☰ to toggle the sidebar.
- **Main panel** (center) — the active tab's content with search, sort, and filter controls at the top.
- **Toolbar** (top) — the app title, **Import** button, and **Export** button.

## Track your first console

<Steps>
  <Step title="Open the Consoles tab">
    Click **Consoles** in the sidebar. You'll see all supported platforms listed in pages of 20 consoles each.
  </Step>
  <Step title="Find your console">
    Use the **Search** box to type part of the console name or manufacturer (e.g., "Nintendo" or "NES"). You can also use the **Sort by** options (Title, Year, Status) or **Filter** (All, Owned, Favorites, Wishlist, Not Owned) to narrow the list.
  </Step>
  <Step title="Mark it as owned">
    In the console card, check the **Owned** checkbox. The change is saved automatically within 500 ms — no save button needed.

    You can also check **Favorite** and **Wishlist** simultaneously if you want to flag it multiple ways.
  </Step>
  <Step title="Add a note (optional)">
    Click in the **Notes** text box on the console card and type anything you want to remember — serial number, condition, where you got it.
  </Step>
</Steps>

## Track your first game

<Steps>
  <Step title="Open the Games tab">
    Click **Games** in the sidebar.
  </Step>
  <Step title="Select a console">
    Use the **Select Console** dropdown at the top of the tab. Choose a specific console to see only its games, or select **All Consoles** to search across every platform at once.
  </Step>
  <Step title="Find a game">
    Type in the **Search** box to filter by title. The search is Unicode-aware — accented characters match their base form (é matches e), and punctuation like apostrophes and colons is ignored, so `Zelda` matches `The Legend of Zelda`.

    You can also sort by **Title**, **Year**, or **Status**, and filter by **All**, **Owned**, **Favorites**, **Wishlist**, or **Not Owned**.
  </Step>
  <Step title="Mark it">
    In the game card, check **Owned**, **Favorite**, and/or **Wishlist**. Add notes in the Notes field. Changes save automatically.
  </Step>
</Steps>

<Tip>
  Games are paginated — 50 per page. If you don't see a title, check you're on the right page or refine your search query.
</Tip>

## Filter and sort reference

Both the Consoles and Games tabs share the same control set:

| Control | Options |
|---------|---------|
| Sort by | Title, Year, Status |
| Filter  | All, Owned, Favorites, Wishlist, Not Owned |
| Search  | Free-text; Unicode-normalized |

**Status sort order**: Owned → Favorite → Wishlist → unmarked

## Export your collection

<Steps>
  <Step title="Click Export in the toolbar">
    The **Export** button is in the top toolbar alongside the app title.
  </Step>
  <Step title="Save the file">
    **Desktop**: a native file save dialog opens. Save as `memory_pak_export.json` (or any name you prefer).

    **Web browser**: a file named `memory_pak_export.json` downloads automatically.
  </Step>
  <Step title="Store it somewhere safe">
    Copy the file to cloud storage, a USB drive, or email it to yourself. This JSON file contains your complete collection state and can be imported back at any time.
  </Step>
</Steps>

<Tip>
  Export regularly — especially before installing a new version of Memory Pak. The export file is the only way to transfer your collection between devices or restore after reinstalling.
</Tip>
