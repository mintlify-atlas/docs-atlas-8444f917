Memory Pak is a cross-platform retro game and console collection tracker built with Rust and egui. Whether you collect Nintendo classics, Sega hardware, Sony handhelds, or toys-to-life figures like Skylanders and LEGO Dimensions, Memory Pak keeps your entire collection organized in one place — with no account required, no internet connection needed, and everything stored locally on your device.

## Key features

- **Owned / Favorite / Wishlist states** — mark any item with one or more statuses and add personal notes
- **Embedded database** — all game data is compiled into the binary, so it works completely offline
- **Cross-console search** — search across all platforms at once with Unicode-aware fuzzy matching
- **Import / Export** — back up your entire collection to a portable JSON file and restore it anywhere
- **Responsive UI** — works on wide desktop screens and narrow mobile displays alike
- **No account needed** — your data stays on your device, in platform-standard user data directories

## Get started

<Steps>
  <Step title="Download or build Memory Pak">
    Grab the latest release for your platform from the [GitHub releases page](https://github.com/Aspenini/Memory-Pak/releases), or follow the [build instructions](/building/overview) to compile from source.
  </Step>
  <Step title="Launch the app">
    Run the binary (or open the installed app). Memory Pak loads its embedded game database automatically — no setup required.
  </Step>
  <Step title="Start marking your collection">
    Use the **Consoles** tab to mark which hardware you own, and the **Games** tab to track your library. Check the [quickstart guide](/quickstart) for a full walkthrough.
  </Step>
  <Step title="Back up your data">
    Use **Export** in the toolbar to save your collection state as JSON. Store it anywhere — Dropbox, a USB drive, or email it to yourself. See [Import / Export](/guides/import-export) for details.
  </Step>
</Steps>
